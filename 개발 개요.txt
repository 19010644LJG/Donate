api 설계 



기부 
- GET 메소드로 기부 페이지를 렌더링
- POST 메소드를 보내면 기부 API

기부 조회
- GET 메소드로 /<id>를 통해 조회
- POST 메소드로 기부 조회 함수 호출 

증명서 발급
- GET 메소드로 contification/<id>
- POST 메소드를 보내면 증명서 발급 함수 호출

웹사이트는 크게 기부, 기부 내역 조회, 증명서 발급 세가지 기능으로 나뉘어요. 개인키 = 고유 번호라고 생각하시면 됩니다.
